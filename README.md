# DNA Sequence Comparison

A project for Design and Analysis of Algorithms class. Compares user-input DNA sequences against a database of known sequences using multiple algorithms and data structures.

## Algorithms
1. **Merge Sort** - Sort sequences by GC content for pre-filtering
2. **Longest Common Subsequence (LCS)** - DNA similarity comparison via dynamic programming
3. **Prim's MST** - Build a minimum spanning tree of sequence relationships

## Data Structures
1. Hash Map - DNA database storage
2. Array/List - Sequence storage and sorting
3. 2D Matrix - LCS dynamic programming table
4. Min-Heap (Priority Queue) - Prim's algorithm
5. Adjacency List (Graph) - Similarity graph and MST

## Tech Stack
- Python + Flask (Web UI)
- matplotlib + networkx (MST visualization)
